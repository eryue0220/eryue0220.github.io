# 2024.01

转眼，2024 年的第一个月就已经过去了。这个月整理了下自己今年的几个想法，发现自己更多的是要做减法而不是做加法。过去的时间里，一直希望自己这也要会，那也要会，不断地学习和尝试新的内容，好处可能是会比其他人都知道些内容，但坏处就是可能都没这么精通。

而且自去年下半年开始花时间在开源项目上后，发现在不同项目间来回切换，除了我以前提到的可以在不同语言来回切换的快乐，副作用也比较明显，自己需要换一天到一周的时间来重新熟悉当前项目的上下文和许多细节，甚至是要怎么重新构建和运行单测都要再看一看。毕竟开源项目的变化还是挺快地，社区每天都有不同人在这上面进行贡献。

## 关于这一年

今年给自己定了几个目标：

- 成为某个开源项目的 maintainer；
- 英语口语能更上一个台阶，希望能够有 B2 的水平；
- 深入的学习 Rust / Golang；
- 学习 Docker / K8s；
- 尝试写一本书或者一个系列的文章；
