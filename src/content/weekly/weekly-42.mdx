## 开源

这周的开源就给 oxc 提交了一个新 [PR](https://github.com/web-infra-dev/oxc/pull/938)，难得的是竟然能够很顺利地就没有在继续和 benchmark 作斗争，很顺利地就合进了 main 分支。果然随着对代码了解程度的增加，以及 Rust 在 Openai 的帮助下，越写越顺手了，代码写起来也比较顺利。在这之外，本周还开了两个新项目，jotai 的 bug 和 opendal 的些迁移相关工作。先说 jotai，一个 react 的 reactive 状态库，代码量虽然不大，但是读起来脑子嗡嗡地，而 bug 的原因作者也给了一个复现的 test case，目前看起来应该是 cache 在某个时刻被 miss 了，导致了某个计算被重复计算了两次，不过因为代码看的太头疼了，暂时没有细看，下周再花些时间看看，具体内容：<https://github.com/pmndrs/jotai/issues/2168> 可以看这里。关于 opendal，这周是在给他们做些 NodeJS 测试用例的更新，目前引入了 vitest 来替换掉之前的 Cucumber.js 测试库，同时还升级了下 service_s3.yml。顺利的话下周应该能把 landing 这个 PR，中年人的第一个 Apache PR 下周应该能达成。

## 远程工作与面试

这周安排了好些面试，但大部分都是国内互联网公司。因为之前投了好些家 remote 岗位，但最终都无疾而终了，所以开始调整了下自己的策略，先养活自己。但这周好像又峰回路转。下周有三家 remote 岗位约面试，以及两家 wfh 的公司，薪资 base 差异都不太大，但就希望能够不要太卷，正常 8 小时应该是自己下一份工作的理想状态，如果再有 remote 的支持，那真是太棒了。体验了三个月的不上班，实在是不想再去坐班了。

面试的话，得益于还算不错得项目经验和一些开源项目的加分项，已经不再怎么有算法面试的这个环节了。不过还是需要有所准备，以防万一。有时候在开发的时候可能也会碰到一些 case 会运用到吧，尽管到目前的开发为止还没有碰到需要算法的地方。唯一被考察最多的就是 react dom 的 diff 算法了吧。

## 英语学习

人的精力果然是有限的，这周英语上的学习就放了下，因为上周读的一本关于英语学习的书，这周开始就在实践下这方面的尝试。除了还是在常规的背单词外，暂时停了很多其他方面的练习或训练，专注于在听的练习上，每天会要求自己在 youtube 上听两个关于非前端相关的技术内容，主要是练习下自己的听力，也在 x 上找些相关的英语 podcast 来听，目前感觉还凑合吧，至少能有 50% ~ 70% 的内容自己能听懂并理解。在逐渐摆脱了”翻译法“的学习后，自己的英语听力确实在有慢慢地提高。毕竟注意力有限，有时候还在关注前面的内容怎么映射为中文时，两句话的时间就过去了。
